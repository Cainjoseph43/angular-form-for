Can be used to configure default behavior for all instances of formFor within a project.

This service can be accessed via $injector.get('FormForConfiguration'). Despite its name, this service is accessible during the Angular run phase (not config).

Static Methods

disableAutoLabels()

Use this method to disable auto-generated labels for formFor input fields.

enableAutoLabels()

Use this method to enable auto-generated labels for formFor input fields. Labels will be generated based on attribute-name for fields without a label attribute present. Radio fields are an exception to this rule. Their names are generated from their values.

disableAutoTrimValues()

Use this method to disable auto-trim.

enableAutoTrimValues()

Use this method to enable auto-trim for input values. Auto-trim will remove leading and trailing whitespace from values before syncing back to the formData object.

getFailedValidationMessage(failureType)

Returns the appropriate error message for the validation failure type.

setDefaultDebounceDuration(duration)

Sets the default debounce interval (in ms) for all textField inputs. This setting can be overridden on a per-input basis (see textField). Defaults to 500ms. To disable debounce (update only on blur) pass false.

setDefaultSubmitComplete(callback)

Sets the default submit-complete behavior for all formFor directives. This setting can be overridden on a per-form basis.

Default handler function accepting a data parameter representing the server-response returned by the submitted form. This function should accept a single parameter, the response data from the form-submit method.

setDefaultSubmitError(callback)

Sets the default submit-error behavior for all formFor directives. This setting can be overridden on a per-form basis (see formFor).

This method should accept a single parameter, the error returned by the form-submit method.

setDefaultValidationFailed(callback)

Sets the default validation-failed behavior for all formFor directives. This setting can be overridden on a per-form basis (see formFor).

setHelpIcon(className)

Sets the CSS class name to be used as the help icon in supported templates. Each template specifies its own default help icon that can be overridden with this method.

setRequiredLabel(string)

Sets a default label to be displayed beside each text and select input for required attributes only.
